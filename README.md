FED4 (Feeding Experimentation Device) is an open-source device for training mice. The device provides comprehensive control over feeding experiments, environmental monitoring, and data collection. Features:

- Automated pellet dispensing system 
- Three nose-pokes
- Pellet approach sensor
- Multi-colored signal LEDs
- Audio feedback
- Haptic feedback
- Temperature/Humidity monitoring
- Activity monitoring
- Optional Hublink integration for network connectivity
